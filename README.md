## How to Run?

1. Python 3.9+ and Pygame must be installed.
2. Navigate to the project folder and run the following command:

```bash
python main.py
```
Controls
1. A → Place a new card
2. 1 / 2 → Player card selection
3. ← / → → Computer card selection
4. SPACE → Attack
5. R → Restart
6. ESC → Exit

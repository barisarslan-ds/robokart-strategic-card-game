## Nasıl Çalıştırılır?

1. Python 3.9+ ve Pygame yüklü olmalıdır.
2. Proje klasörüne gidin ve aşağıdaki komutu çalıştırın:

```bash
python main.py
```

Kontroller
1. A → Yeni kart yerleştir
2. 1 / 2 → Oyuncu kart seçimi
3. ← / → → Bilgisayar kart seçimi
4. SPACE → Saldırı
5. R → Yeniden başlat
6. ESC → Çıkış
